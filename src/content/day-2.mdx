## Day 2: The Building Blocks — HTML Basics & Page Structure

### What is HTML?

HTML stands for **HyperText Markup Language**. It’s the **backbone** of every web page — the language that tells the browser what content to show and how to structure it.

Think of HTML as the **skeleton** of a website.

A basic HTML file looks like this:

<CodeBlock language="html">
{`<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>`}
</CodeBlock>

<Note>
HTML is not a programming language. It doesn’t “do” things — it just tells the browser **what is on the page**.
</Note>

<Divider />

### 🧱 HTML = Tags + Elements

HTML is made up of **tags**. Tags come in pairs — an opening tag and a closing tag.

#### Example:
```html
<p>This is a paragraph.</p>
```

### Understanding the Basic Page Structure

Every HTML document starts with a **boilerplate** — a standard structure that all HTML pages follow.

Let’s break it down:

### 🧠 What each part means:

<Table
  columns={["Part", "Description"]}
  data={[
    ["`<!DOCTYPE html>`", "Tells the browser you’re using HTML5"],
    ["`<html>`", "The root element that wraps all content"],
    ["`<head>`", "Contains metadata like title, fonts, etc. (not visible on page)"],
    ["`<body>`", "Contains the visible content shown in the browser"],
  ]}
/>

<Note>
The browser reads this structure **top to bottom**, so make sure your page is structured properly!
</Note>

<Divider />

### Nesting Elements

In HTML, you can **nest** elements inside one another.

#### Example:

```html
<div>
  <h2>About Me</h2>
  <p>I’m learning web design!</p>
</div>
```
### Creating a Basic HTML Page

Let’s now apply what we’ve learned and build a basic HTML page structure. This will include all the important parts: `<!DOCTYPE>`, `<html>`, `<head>`, and `<body>`.

<CodeBlock language="html">
{`<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first web page.</p>
  </body>
</html>`}
</CodeBlock>

### Common HTML Elements You’ll Use a Lot

Here are some essential HTML tags you'll find yourself using all the time. We'll explore them more in later days, but here's a quick taste.

<Table
  columns={["Tag", "Purpose", "Example"]}
  data={[
    ["`<h1>` to `<h6>`", "Headings of different sizes", "`<h1>Main Heading</h1>`"],
    ["`<p>`", "Paragraph", "`<p>This is a paragraph.</p>`"],
    ["`<a>`", "Anchor (Link)", "`<a href='https://google.com'>Visit Google</a>`"],
    ["`<img>`", "Image", "`<img src='logo.png' alt='My Logo' />`"],
    ["`<ul>` / `<ol>`", "Unordered / Ordered List", "`<ul><li>Item</li></ul>`"],
    ["`<div>`", "Generic container", "`<div>Content here</div>`"]
  ]}
