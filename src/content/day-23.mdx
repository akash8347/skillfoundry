## 🧾 Day 23: Form Handling in JavaScript

Forms are how users interact with your site — whether it's signing up, logging in, or submitting a message. Today, we’ll learn how to work with form inputs, capture their values, validate them, and respond to user actions using **JavaScript**.

<Note>
This is a foundational topic. Mastering forms will help you in every web project that involves user input.
</Note>

<Divider />

## 🧠 What is a Form?

A form is a container for input elements that collect user data and submit it. Here's a simple example:

<CodeBlock language="html">
{`<form id="contactForm">
  <input type="text" id="name" placeholder="Your Name" />
  <button type="submit">Send</button>
</form>
<div id="output" style="margin-top: 1rem; font-weight: bold;"></div>`}
</CodeBlock>

<CodeBlock language="js">
{`window.onload = function () {
  document.getElementById("contactForm").addEventListener("submit", function(e) {
    e.preventDefault();
    document.getElementById("output").innerHTML = "Form submitted!";
  });
};`}
</CodeBlock>

## 🧩 How Form Inputs Work

Each form input element has a `.value` property in JavaScript. Here's how you can get what the user typed:

<CodeBlock language="html">
{`<input type="text" id="username" placeholder="Type something..." />
<button onclick="greet()">Say Hi</button>
<div id="output"></div>`}
</CodeBlock>

<CodeBlock language="js">
{`function greet() {
  const name = document.getElementById("username").value;
  document.getElementById("output").innerHTML = "Hello, " + name + "!";
}`}
</CodeBlock>

## 🛠️ Building a Real Registration Form

Let’s create a full form with **Name**, **Email**, **Password**, and a **Subscribe** checkbox. We'll validate the inputs before allowing submission.

<CodeBlock language="html">
{`<form id="registerForm">
  <input type="text" id="name" placeholder="Full Name" /><br />
  <input type="email" id="email" placeholder="Email Address" /><br />
  <input type="password" id="password" placeholder="Password (min 6 chars)" /><br />
  <label>
    <input type="checkbox" id="subscribe" />
    Subscribe to Newsletter
  </label><br /><br />
  <button type="submit">Register</button>
  <div id="error" style="color:red;"></div>
</form>`}
</CodeBlock>

<CodeBlock language="js">
{`document.getElementById("registerForm").addEventListener("submit", function(e) {
  e.preventDefault();

  const name = document.getElementById("name").value.trim();
  const email = document.getElementById("email").value.trim();
  const password = document.getElementById("password").value;
  const error = document.getElementById("error");

  if (name === "" || email === "" || password === "") {
    error.textContent = "All fields are required.";
    return;
  }

  if (!email.includes("@")) {
    error.textContent = "Please enter a valid email.";
    return;
  }

  if (password.length < 6) {
    error.textContent = "Password must be at least 6 characters.";
    return;
  }

  error.textContent = "";
  alert("Registration successful!");
});`}
</CodeBlock>

## 🎯 What Happens Behind the Scenes?

When the form is submitted:

1. JavaScript **prevents default submission** with `e.preventDefault()`.
2. It **reads values** using `.value`.
3. Checks conditions like `password.length < 6`.
4. Displays messages using `.textContent`.

This is the heart of client-side form handling.

<Note>
We're not sending data anywhere yet. We'll cover actual submission using `FormData` or `fetch()` in future lessons.
</Note>

## ✨ Inline Error Messages & Real-Time Validation

A great form doesn’t just reject input — it helps users fix it as they type. Let’s show error messages **under each field**, and make them disappear when corrected.

<CodeBlock language="html">
{`<form id="signupForm">
  <input type="text" id="username" placeholder="Username" />
  <div id="nameError" style="color:red;"></div>

  <input type="email" id="useremail" placeholder="Email" />
  <div id="emailError" style="color:red;"></div>

  <button type="submit">Submit</button>
  <div id="success" style="color: green; font-weight: bold;"></div>
</form>`}
</CodeBlock>

<CodeBlock language="js">
{`document.getElementById("signupForm").addEventListener("submit", function(e) {
  e.preventDefault();

  const nameInput = document.getElementById("username");
  const emailInput = document.getElementById("useremail");
  const nameError = document.getElementById("nameError");
  const emailError = document.getElementById("emailError");
  const success = document.getElementById("success");

  nameError.textContent = "";
  emailError.textContent = "";
  success.textContent = "";

  let hasError = false;

  if (nameInput.value.trim() === "") {
    nameError.textContent = "Name is required.";
    hasError = true;
  }

  if (!emailInput.value.includes("@")) {
    emailError.textContent = "Invalid email address.";
    hasError = true;
  }

  if (!hasError) {
    success.textContent = "Form submitted successfully!";
  }
});`}
</CodeBlock>

## ✅ Handling Checkboxes

Checkboxes return `true` or `false` using `.checked`.

<CodeBlock language="html">
{`<label>
  <input type="checkbox" id="terms" />
  I agree to the terms and conditions
</label>
<br />
<button onclick="checkTerms()">Submit</button>
<div id="result" style="font-weight: bold; margin-top: 1rem;"></div>`}
</CodeBlock>

<CodeBlock language="js">
{`function checkTerms() {
  const isChecked = document.getElementById("terms").checked;
  const result = document.getElementById("result");

  if (isChecked) {
    result.style.color = "green";
    result.textContent = "Thank you for agreeing!";
  } else {
    result.style.color = "red";
    result.textContent = "You must agree to proceed.";
  }
}`}
</CodeBlock>

## 🔘 Handling Radio Buttons

Radio buttons are grouped by the `name` attribute. You can find the selected one like this:

<CodeBlock language="html">
{`<label><input type="radio" name="color" value="Red" /> Red</label>
<label><input type="radio" name="color" value="Green" /> Green</label>
<label><input type="radio" name="color" value="Blue" /> Blue</label>
<br />
<button onclick="getColor()">Submit</button>
<div id="result" style="font-weight: bold; margin-top: 1rem;"></div>`}
</CodeBlock>

<CodeBlock language="js">
{`function getColor() {
  const selected = document.querySelector('input[name="color"]:checked');
  const result = document.getElementById("result");

  if (selected) {
    result.style.color = "green";
    result.textContent = "You selected: " + selected.value;
  } else {
    result.style.color = "red";
    result.textContent = "Please select a color.";
  }
}`}
</CodeBlock>

## ⬇️ Handling Select Dropdowns

With dropdowns, you just grab the `.value` of the selected option.

<CodeBlock language="html">
{`<label htmlFor="lang">Choose a language:</label>
<select id="lang">
  <option value="">-- Select --</option>
  <option value="HTML">HTML</option>
  <option value="CSS">CSS</option>
  <option value="JavaScript">JavaScript</option>
</select>
<br />
<button onclick="getLanguage()">Submit</button>
<div id="output" style="margin-top: 1rem; font-weight: bold;"></div>`}
</CodeBlock>

<CodeBlock language="js">
{`function getLanguage() {
  const value = document.getElementById("lang").value;
  const output = document.getElementById("output");

  if (value) {
    output.textContent = "You selected: " + value;
    output.style.color = "green";
  } else {
    output.textContent = "Please choose a language first.";
    output.style.color = "red";
  }
}`}
</CodeBlock>

- Validate user input and show inline error messages.
