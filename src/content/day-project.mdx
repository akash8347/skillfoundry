## 📖 About This Project Collection

This collection contains **100+ focused projects** tailored for beginner to intermediate learners in the Skill Foundry ecosystem.

These projects are structured to sharpen your understanding of:

- 🧱 DOM Manipulation
- ⚡ ES6+ Features
- 🔗 API Integration
- 🧠 Data Structures & Algorithms
- 🖱️ Event Handling
- ⏳ Asynchronous JavaScript (Promises, Async/Await)
- 💾 Browser Storage (LocalStorage, SessionStorage)
- 💡 And many more practical use-cases!

Each project is built around a specific concept. Whether you're practicing form validation, fetching APIs, or animating elements with CSS and JS, there’s a bit of everything to help level up your skills.

<Tip>
These projects aren’t just for practice. You can use them to build your portfolio or reuse the code in your freelance/side projects too!
</Tip>

## ⚙️ How to Use These Projects

Follow these simple steps to start experimenting and coding:

<CodeBlock language="bash">
git clone https://github.com/pradipchaudhary/100-js-projects.git
</CodeBlock>

<CodeBlock language="bash">
cd 100-js-projects
</CodeBlock>

Each project lives inside its own folder and includes an `index.html` file to help you get started quickly.

<BrowserMockup>
Open <code>index.html</code> directly in your browser or launch it via a local dev server like VSCode Live Server.
</BrowserMockup>
